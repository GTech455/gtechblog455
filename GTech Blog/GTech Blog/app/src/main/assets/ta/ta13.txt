Olá. Nesse tutorial vou ensinar como criar Gifs com Gif Creator. Essa ferramenta é um criador de Gifs instantaneo que pode lhe ofereçer diversos recursos. Bom, após aberto toque no ícone + do cabeçalho do App para criar novo Gif, em seguida using Gallery ( Usando a Galeria ). Escolha a imagem desejada, uma a uma. O gif é um conjunto de imagens de diferêntes movimentos. Após isso, toque no ícone concluir, abaixo de uma velocidade para seu gif. Pronto! O Gif foi criado com êxito.
